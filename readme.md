# Tiep Phan Blog
This blog built with Hugo, PostCSS, Tailwindcss

## Development

### Install dev dependencies
```sh
npm install
```
### Dev Hugo
```sh
hugo server
```
### Build SSG
```sh
npm run build
```

Build with other environment
```sh
npm run build -- -e <production | next>
```
```sh
npm run build -- -e next
```
